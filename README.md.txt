# InPursuit Health Website (Static Version)

This is the static HTML site for InPursuit Health — healthcare data interoperability, TETRA platform, value-based care models, MAHA alignment, and patient-owned data focus.

## How to View Locally
- Open any `.html` file in your browser (no server needed).

## Navigation & Footer
- Shared header + footer are injected automatically via `nav.js`.
- Mobile menu, scroll shadow, active link highlighting included.

## Deploy to GitHub Pages (free hosting)
1. Push this repo to GitHub.
2. Go to repo Settings → Pages.
3. Set Source: Deploy from a branch → main → / (root).
4. Wait 1–2 minutes → site live at https://yourusername.github.io/inpursuit-health-website/
